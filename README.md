# Graph Projekt

Dieses Projekt implementiert einen Graphen in Java.

## Funktionen

- CSV-Import
- Adjazenzmatrix
- BFS (Breadth First Search)
- DFS (Depth First Search)
- Berechnung von:
  - Distanzen
  - Exzentrizitäten
  - Radius
  - Durchmesser
  - Zentrum

## Starten

Projekt in IntelliJ öffnen und die Klasse `Main` ausführen.
